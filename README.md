# s3mplay
A music player library for S3M tracker music along with a litte player application

## How to build

### Prerquisites

SDL2 development library.

### Build steps

1. Create build directory

    mkdir build
    cd build

2. Run CMake
 
    cmake ../src

3. Build it
 
    make
